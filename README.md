Sudoku
======
In Haskell.

Dependencies
============
Main dependencies: 
* Haskell (>= 7.6.1)
* ncurses (~> 0.2.14)
* lens (~> 4.13)
* lens-simple (~> 0.1.0.8)
* MissingH (~> 1.3.0.1)

Other dependencies: please see `sudoku.cabal`.

How to build
============
Use `cabal build`.

After this stage, the executable should be locate at `dist/build/sudoku/sudoku`,
and if not, please check the cabal output.
